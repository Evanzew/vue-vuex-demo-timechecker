## Database Structure

> Task structure

```bash
    "Id" : 0,
    "User_Name" : "Colin.Miao",
    "Date" : ISODate("2017-12-31T08:50:20.201Z"),
    "Time" : 1,
    "Comments" : "看直播一小时"
```

> User structure

```bash
  "User_Name" : "Evan.Zou",
  "Password" : "123"
```

## Commissioning

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```
